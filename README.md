# To-do-list


## Description
A console-based to-do list in C that saves tasks to a file using file handling, allowing persistent storage.

## Features
- Add tasks
- View all tasks
- Tasks are saved to a file and persist after program exits

## How to Run
1. Open terminal/command prompt
2. Navigate to the To-Do-List folder
3. Compile using: gcc todo_list.c -o todo_list
4. Run using: ./todo_list

## Technologies Used
- C programming
- File handling in C
